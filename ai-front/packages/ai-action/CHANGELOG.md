## [1.0.0] - 2025-01-XX

### Added

- Initial release of @maxpro/ai-action
- Dual mode support: hover mode and standalone button mode
- Draggable modal window with AI assistant iframe
- TypeScript support with full type definitions
- Customizable base URL for AI assistant endpoint
- Accessibility support with ARIA labels
- Custom icon slot support
- Event emission on button click
- Comprehensive documentation in English and Chinese
